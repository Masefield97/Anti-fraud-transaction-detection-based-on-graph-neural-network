# 基于图神经网络的反欺诈交易检测
针对金融交易中欺诈行为隐蔽性强、团伙化作案难以识别的问题，设计并实现了一套基于图神经网络GraphSAGE框架的端到端反欺诈检测系统。以Elliptic 公开比特币交易数据集为研究对象，该数据集包含 203,769 个交易节点 和 234,355 条资金流向边，其中欺诈节点占比约 2.1%（约 4,279 个）。将账户建模为图节点，交易关系建模为边，构建大规模有向交易网络图，利用拓扑结构提升对稀疏欺诈样本的识别能力。
